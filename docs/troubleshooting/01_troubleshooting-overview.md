# トラブルシューティング

開発中に発生するよくある問題と、その解決策をまとめます。

## ドキュメント一覧

| No  | ドキュメント                | 内容                                  |
| --- | --------------------------- | ------------------------------------- |
| 01  | 概要（本ドキュメント）      | トラブルシューティングの全体案内      |
| 02  | 環境構築トラブル            | 環境構築時によくあるエラーと解決策    |
| 03  | Next.js トラブル            | ビルド・レンダリングエラーの対処      |
| 04  | Supabase トラブル           | 接続・RLS・エッジ関数のトラブル       |
| 05  | Cloudflare デプロイトラブル | Workers / OpenNext デプロイ時のエラー |

## 解決の流れ

1. エラーメッセージを確認し、本ドキュメントの該当セクションを参照
2. 社内チャット / Issue で同様の事例を検索
3. 解決できない場合はチームリーダーにエスカレーション
